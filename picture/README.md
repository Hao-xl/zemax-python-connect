这个文件夹用于在发布到 GitHub 前存放给用户查看的截图。

建议文件名：

- `interactive-extension-button.png`：OpticStudio `Programming / 编程` 选项卡截图，重点显示 `ZOS-API.NET` 区域中独立的 `交互扩展 / Interactive Extension` 按钮。
- `interactive-extension-waiting-dialog.png`：交互扩展等待连接对话框截图，截图中应能看到 `Instance Number`、等待连接状态、`Auto Close on Disconnect`、`Terminate` 和 `Close`。

`SKILL.md` 和 `README.md` 会引用这些截图，帮助用户确认自己已经正确启动交互扩展模式。如果实际图片文件名不同，请同步修改 `README.md` 中的图片链接。
